## 🚀 使用建议

### 开启动画
所有动画已自动启用，无需额外配置。

### 性能调优
如果遇到性能问题，可以：
1. 减少 `particleCount` (在 EnhancedWaterWave.jsx 中)
2. 降低 Three.js Canvas 的 `dpr` 参数
3. 减少 FloatingElements 中的装饰元素数量

### 自定义动画
- 修改 `tailwind.config.js` 中的颜色
- 调整动画 duration 和 ease 参数
- 添加新的 CSS 动画关键帧

## 🎯 动画哲学

我们的动画设计遵循以下原则：
1. **有意义** - 每个动画都有目的，增强用户理解
2. **自然** - 使用弹簧物理和缓动函数
3. **流畅** - 60fps 的丝滑体验
4. **克制** - 避免过度动画导致注意力分散
5. **愉悦** - 微交互带来的惊喜感
