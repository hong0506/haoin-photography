# About页面重新设计说明

## 🎨 设计理念

全新的About页面以**程序员、创业者、摄影爱好者**三重身份为核心，展现一个真实、立体、充满活力的个人形象。

---

## ✨ 核心特点

### 1. 真实的个人故事
- ✅ 不再是虚构的"专业摄影师"身份
- ✅ 强调程序员和创业者的主业
- ✅ 摄影作为热爱的副业
- ✅ 真实的旅行经历和足迹

### 2. 丰富的动画效果
- 🎭 **视差滚动** - 背景元素呼吸式动画
- 🎪 **悬停交互** - 所有卡片都有精心设计的hover效果

- 🌈 渐变色彩系统

## 🎭 动画类型总览

### 入场动画
```javascript
- fadeIn: 淡入
- slideUp: 上滑进入
- slideFromLeft: 从左滑入
- slideFromRight: 从右滑入
- scaleIn: 缩放进入
- staggered: 错峰依次进入
```

### 交互动画
```javascript
- hover:scale: 悬停放大
- hover:y-lift: 悬停抬起
- hover:rotate: 悬停旋转
- hover:glow: 悬停发光
- number-jump: 数字跳动
- icon-spin: 图标旋转
```

### 持续动画
```javascript
- pulse: 脉冲呼吸
- float: 上下浮动
- rotate-loop: 循环旋转
- gradient-shift: 渐变流动
```

```javascript
  icon: "旋转12度",
```

### 3. 国家标签
```javascript
// 依次弹出
transition: {
  delay: index * 0.05,
  type: "spring",
  stiffness: 300

  scale: 1.15,
  rotate: 5
}
```

### 4. 时间线图标
